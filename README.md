# Расписание экзаменов — прототип (React + Vite)

Небольшое клиентское приложение для управления расписанием экзаменов с ролями: ученик, преподаватель и администратор. Это прототип: данные хранятся в памяти и в локальном хранилище (localStorage). Проект собран на Vite + React.

## Что внутри

- `index.html` — шаблон страницы и подключение скрипта.
- `src/main.jsx` — точка входа React.
- `src/App.jsx` — основной компонент приложения (авторизация, роли, создание и редактирование экзаменов).
- `src/App.css` — стили, поддержка светлой/тёмной темы.
- `public/` — статические ресурсы.

## Возможности

- Локальная авторизация с тестовыми аккаунтами.
- Роли: `student`, `teacher`, `admin`.
- Создание экзаменов: класс, предмет, дата, время, список учеников.
- Редактирование экзаменов в таблице (для преподавателей/админов).
- Карточный вид для учеников, фильтр по классу, сортировка, печать.
- Тёмная и светлая темы с сохранением в `localStorage`.
- Экзамены сохраняются в `localStorage`.

## Тестовые учётные записи (локально)

- Иван — `student` — login: `ivan` / password: `1234`
- Мария — `teacher` — login: `maria` / password: `5678`
- Админ — `admin` — login: `admin` / password: `admin`

## Быстрый запуск (локально)

Требуется Node.js и npm.

1. Установить зависимости:

```powershell
npm install
```

2. Запустить dev-сервер:

```powershell
npm run dev
```

3. Открыть адрес, который выведет Vite (обычно `http://localhost:5173`).

## Как проверить базовые сценарии

- Войти под `ivan` — увидеть карточки своих экзаменов.
- Войти под `maria` — добавить/редактировать/удалять экзамены через таблицу.
- Переключить тему в шапке — тема сохраняется.
